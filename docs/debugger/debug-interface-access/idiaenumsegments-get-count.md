# <a name="idiaenumsegmentsget_count"></a>IDiaEnumSegments::get_Count
检索段的数量。

## <a name="syntax"></a>语法

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>参数
 pRetVal
- [out, retval] 返回段的数量。

## <a name="return-value"></a>返回值
 如果成功，则返回 `S_OK`；否则，返回错误代码。
