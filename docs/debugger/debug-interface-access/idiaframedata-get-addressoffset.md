# <a name="idiaframedataget_addressoffset"></a>IDiaFrameData::get_addressOffset
检索帧的代码地址的偏移量部分。

## <a name="syntax"></a>语法

```C++
HRESULT get_addressOffset ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out] 返回帧的代码地址的偏移量部分。

## <a name="return-value"></a>返回值
 如果成功，则返回 `S_OK`。 如果不支持此属性，则返回 `S_FALSE`。 否则，返回错误代码。
