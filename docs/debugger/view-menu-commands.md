# <a name="view-menu-commands"></a>“视图”菜单命令
“视图”菜单提供用于控制 Spy++ 用户界面的命令。

 字体 打开“字体”对话框，可以在其中选择字体以用于当前 Spy++ 视图。

 工具栏 隐藏或显示工具栏。

 状态栏 隐藏或显示状态栏。

 属性 打开活动视图的“属性”对话框。
