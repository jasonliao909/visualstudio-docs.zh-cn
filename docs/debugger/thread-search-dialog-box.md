# <a name="thread-search-dialog-box"></a>“线程搜索”对话框
使用此对话框可在[线程视图](../debugger/threads-view.md)中查找并选择特定线程的节点。 若要显示此对话框，请将焦点移至线程视图窗口。 然后从“搜索”菜单中选择“查找线程”。

 提供了下列设置：

 **线程**：要搜索的线程 ID。

 **模块**：要搜索的模块字符串。

 **向上或向下搜索** 搜索的初始方向。
