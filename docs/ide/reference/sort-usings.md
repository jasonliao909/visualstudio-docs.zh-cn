# <a name="sort-usings"></a>对 using 排序

此重构适用于：

- C#

- Visual Basic

内容：对 using 排序。

适用情况：需要对文件顶部的 `using` 指令排序，让它们按字母顺序排序。 

原因：这样便于查找 using 指令。

## <a name="how-to"></a>操作方法

1. 从菜单栏中选择“编辑”。
2. 选择“Intellisense” > “对 Using 排序” 。
