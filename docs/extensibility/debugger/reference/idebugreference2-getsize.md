# <a name="idebugreference2getsize"></a>IDebugReference2::GetSize
获取引用值的大小（以字节为单位）。 保留供将来使用。

## <a name="syntax"></a>语法

```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```

```csharp
int GetSize ( 
   out uint pdwSize
);
```

## <a name="parameters"></a>parameters
`pdwSize`\
[out]返回引用值的大小（以字节为单位）。

## <a name="return-value"></a>返回值
 始终返回 `E_NOTIMPL`。
