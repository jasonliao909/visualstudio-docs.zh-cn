# <a name="process-debug-manager"></a>进程调试管理器
PDM (进程) 是 Visual Studio 组件的一部分，用于管理程序和进程，使其可用于会话调试管理器和调试引擎。

 PDM 管理所有可调试的进程。 若要进行调试，必须将程序注册到 PDM。 此注册是在启动程序时通过端口或调试引擎完成。
