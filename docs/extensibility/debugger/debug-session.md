# <a name="debug-session"></a>调试会话
就调试器体系结构而言，会话 *：*

- 从 IDE 的单个实例启动或附加到的所有进程（相关或无关）的集。

- 由会话调试管理器 (SDM) 。 有关详细信息，请参阅会话 [调试管理器](../../extensibility/debugger/session-debug-manager.md)。
