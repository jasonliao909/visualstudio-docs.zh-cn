# <a name="menu-element"></a>Menu 元素
定义一个菜单项。 以下是六种类型的菜单： Context、Menu、MenuController、MenuControllerLatched、Toolbar 和 ToolWindowToolbar。

## <a name="syntax"></a>语法

```xml
<Menu guid="guidMyCommandSet" id="MyCommand" priority="0x100" type="button">
  <Parent>... </Parent>
  <CommandFlag>... </CommandFlag>
  <Strings>... </Strings>
</Menu>
```

## <a name="attributes-and-elements"></a>特性和元素
 下列各节描述了特性、子元素和父元素。

|priority|可选。 一个数字值，该值指定菜单在一组菜单中的相对位置。|
|toolbarPriorityInBand|可选。 一个数值，该数值指定工具栏在停靠窗口时的相对位置。|

### <a name="child-elements"></a>子元素

|元素|说明|
|-------------|-----------------|
|Parent|可选。 菜单项的父元素。|
|CommandFlag|必需。 请参阅 [命令标志元素](../extensibility/command-flag-element.md)。 菜单的有效 CommandFlag 值如下所示：<br /><br /> -   **AlwaysCreate**<br />-   **DefaultDocked**<br />-   **DefaultInvisible** - 此标志不影响工具栏的显示。<br />-   **DontCache**<br />-   **DynamicVisibility** - 此标志不影响工具栏的显示。<br />-   **IconAndText**<br />-   **NoCustomize**<br />-   **NotInTBList**<br />-   **NoToolbarClose**<br />-   **TextChanges**<br />-   **TextIsAnchorCommand**|
|字符串|必需。 请参阅 [Strings 元素](../extensibility/strings-element.md)。 必须 `ButtonText` 定义子元素。|
|Annotation|可选注释。|

### <a name="parent-elements"></a>父元素

|元素|说明|
|-------------|-----------------|

## <a name="example"></a>示例

```
<Menu guid="cmdGuidWidgetCommands" id="menuIDEditWidget"
  priority="0x0002" type="Menu">
  <Parent guid="cmdSetGuidWidgetCommands" id="groupIDFileEdit"/>
  <CommandFlag>AlwaysCreate</CommandFlag>
  <Strings>
    <ButtonText>Edit Widget</ButtonText>
  </Strings>
</Menu>
```
