# <a name="msb6001-invalid-command-line-switch-for-tool"></a>MSB6001：“tool”的命令行开关无效

如果为 MSBuild 任务指定的命令行选项无效，则会出现此错误。 查看第二个错误，了解有关该问题的更多详细信息。

常见原因包括：

- 路径中有非法字符。 请检查引号的位置，确保它们未被不适当地使用。
- 开关的语法中存在拼写错误
- 你正在使用的工具的版本不支持该开关
