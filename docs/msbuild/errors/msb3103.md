# <a name="msb3103-invalid-resx-file"></a>MSB3103：Resx 文件无效

当 .resx 资源文件出现问题时，会发生此错误。 当你看到此错误时，会显示另一个包含更多详细信息的错误。 有关详细信息，请查看另一个错误。
