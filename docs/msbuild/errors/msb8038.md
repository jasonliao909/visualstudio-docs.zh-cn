# <a name="msbuild-error-msb8038"></a>MSBuild 错误 MSB8038

> MSB8038：未定义平台工具集。 请在“项目属性”UI 中选择一个可用的平台工具集。

若要解决此问题，请修改项目以指定平台工具集。 有关详细信息，请查看[平台工具集](/cpp/build/reference/general-property-page-project#platform-toolset)。
