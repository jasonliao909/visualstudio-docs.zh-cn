# <a name="msbuild-error-msb8000"></a>MSBuild 错误 MSB8000

> MSB8000：“name”是“Type”参数的无效值。 有效值为：Warning 和 Error。

若要解决此问题，请对 `Type` 参数使用 `Warning` 或 `Error`。
