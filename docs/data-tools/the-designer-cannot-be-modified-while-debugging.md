# <a name="the-designer-cannot-be-modified-while-debugging"></a>调试时无法修改设计器

应用程序以调试模式运行时，如果试图对 O/R 设计器中的项进行修改，则会出现此消息。 应用程序以调试模式运行时，O/R 设计器是只读的。

若要更正此错误，请在“调试”菜单上选择“停止调试” 。 应用程序将停止调试，从而可以对 O/R 设计器中的项进行修改。
