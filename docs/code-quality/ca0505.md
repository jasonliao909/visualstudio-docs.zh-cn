# <a name="ca0505"></a>CA0505

将忽略已弃用的属性，因为已定义替代属性。

当已弃用的代码分析属性和替代代码分析属性在项目或生成配置文件中都有值时，将生成警告 CA505。 若要解决此问题，请删除已弃用的属性。
