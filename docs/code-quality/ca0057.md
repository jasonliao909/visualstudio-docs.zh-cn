# <a name="ca0057"></a>CA0057

无法将冲突映射到当前目标和规则集

如果 FxCopCmd.exe 是使用项目文件或具有无法映射到当前目标和规则集的冲突的/imported XML 报表手动调用的，则会发生此错误。

仅当使用工具的命令行版本进行分析时才会发生。
