# <a name="ca0055"></a>CA0055

无法加载文件

代码分析工具无法加载为分析指定的目标文件。 如果找不到文件或访问文件被拒绝，则会发生这种情况。 通常情况下，如果用户选择在 Visual Studio 中重写输入程序集属性，或手动调用了 fxcopcmd.exe，就会发生这种情况。
