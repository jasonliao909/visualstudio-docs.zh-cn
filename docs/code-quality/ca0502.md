# <a name="ca0502"></a>CA0502

不支持的语言

当用户尝试分析 ASP.NET 包含一个或多个用 C# 或 Visual Basic 语言编写的页面或类的一个或多个项目时，会发生Visual Basic。 若要修复此错误，请从项目中删除文件或禁用代码分析。
