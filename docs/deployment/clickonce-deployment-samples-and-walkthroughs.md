# <a name="clickonce-deployment-samples-and-walkthroughs"></a>ClickOnce 部署示例和演练
本部分包含示例应用程序、示例代码和分步演练，它们反映了用于部署 Windows 窗体、WPF 和控制台应用程序的语法、结构和技术。

 此示例代码仅供教学使用，不应在未修改的情况下用于已部署的解决方案。 特别要注意的是安全性。
