## <a name="visual-studio-core-editor-included-with-visual-studio-team-explorer-2022"></a>Visual Studio 2022 (2022 Visual Studio 团队资源管理器 中包含的内核编辑器) 

**ID：** Microsoft.VisualStudio.Workload.CoreEditor

**描述：** Visual Studio 核心 shell 体验，包括语法感知代码编辑、源代码管理和工作项管理。

### <a name="components-included-by-this-workload"></a>此工作负载所包含的组件

组件 ID | “属性” | Version | 依赖项类型
--- | --- | --- | ---
Microsoft.VisualStudio.Component.CoreEditor | Visual Studio 核心编辑器 | 17.1.32112.364 | 必需
