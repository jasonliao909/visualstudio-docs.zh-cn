# <a name="visual-studio-subscriptions-offered-to-startups"></a>向初创企业提供的 Visual Studio 订阅

加入 Microsoft for Startups 计划的合格初创公司可以赚取 5 Visual Studio Enterprise [每月云订阅](https://visualstudio.microsoft.com/vs/pricing-details/)。 

## <a name="subscription-details"></a>订阅详细信息 

这些订阅与通过Azure 市场提供的订阅相同，没有任何特殊限制或限制。 必须先将每个订阅分配给最终用户，然后用户才能运行Visual Studio，并在Azure DevOps Services中被识别为Visual Studio Enterprise订阅者。

> [!Note]
> 对于所有 Visual Studio 订阅，许可用户都必须使用激活 Visual Studio 订阅时所用的相同标识/登录名登录到 Visual Studio。

产品密钥未提供用于解锁产品;登录是运行Visual Studio的唯一选项。 在 Visual Studio 停止工作之前，Visual Studio 订阅者可脱机工作长达 30 天。 当用户联机时，Visual Studio以无提示方式刷新用户的许可证状态，以确保订阅仍然有效。

## <a name="frequently-asked-questions"></a>常见问题

### <a name="q-what-benefits-are-included-with-subscriptions-provided-through-microsoft-for-startups"></a>问：通过 Microsoft for Startups 提供的订阅包含哪些权益？

答：这些订阅是每月订阅，权益有限。  订阅包括：
+ Visual Studio IDE for Windows 和 Mac
+ Azure DevOps 
+ Azure DevOps Server 有关详细信息，请参阅我们的[定价详细信息页](https://visualstudio.microsoft.com/vs/pricing-details/)
